Matplotlib Cheat Sheet

从 `matplotlib cheat.pdf` 提炼的常用 API 速查（v3.5.0）
来源：`/Users/junzhang/Downloads/matplotlib cheat.pdf`

快速起手

  • import matplotlib.pyplot as plt
  • fig, ax = plt.subplots()
  • ax.plot(X, Y) / ax.scatter(X, Y)
  • fig.savefig("figure.pdf"), plt.show()
  • fig.tight_layout() 做自动布局

高频绘图 API

  • plot, scatter, bar/barh, hist
  • imshow, contour/contourf, pcolormesh, hexbin
  • errorbar, boxplot, violinplot, eventplot
  • pie, quiver, fill_between, step

坐标与比例尺

  • ax.set_xscale(...) / ax.set_yscale(...)
  • 可选：linear, log, symlog, logit
  • ax.set_xlim/ylim, ax.set_xticks/yticks
  • projection="polar" / projection="3d"

样式与颜色

  • 线型："-" ":" "--" "-."
  • 颜色："C0"、命名色、(R,G,B[,A])、#RRGGBB[AA]
  • colormap：plt.get_cmap(name)，反转用 _r
  • plt.style.use("ggplot") / "fast" / "seaborn"

Ticks / Formatters

  • from matplotlib import ticker
  • locator：AutoLocator, MultipleLocator, LogLocator
  • formatter：ScalarFormatter, FuncFormatter, PercentFormatter
  • ax.xaxis.set_major_locator(...) / set_major_formatter(...)

注释、图例、颜色条

  • ax.legend(...)：位置、边框、间距等
  • ax.colorbar(...)：连续值映射
  • ax.annotate(text, xy=..., xytext=...)：箭头注释
  • 连接样式：arc3, angle, bar 等

性能提示

  • 大量点：plot(..., marker="o", ls="") 往往比 scatter 更快
  • 动画更新优先 set_data，避免反复 cla() + 重绘
  • 批量绘线可先拼接数据，减少 Python 循环开销

键盘快捷（Matplotlib 窗口）

  • p 平移，o 框选缩放，r 复位
  • f 全屏，ctrl+s 保存，ctrl+w 关闭
  • g/G 切换网格，l/L 切换 log/linear
